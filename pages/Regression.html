What is Multiple Linear Regression?


Multiple Linear Regression is allows an analyst to describe the relationship between multiple independent variables to predict a dependent variable (movie ratings).



Why use Multiple Linear Regression?

Multiple linear regression is best used when there is a strong relationship between two or more indepentent variables and one dependent variable. It is best used with numerical data.
